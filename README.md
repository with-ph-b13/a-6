# DigiTools

## Project Overview

Digitools is a web application that provides users to buy premium digital tools, design assets, templates, and productivity software.

## Features

- User can buy premium digital tools
- User can buy design assets
- User can buy templates
- User can buy productivity software
- User can add products to cart
- User can remove products from cart
- User can checkout
- User can view cart
- User can view products
- User can view steps
- User can view pricing
- User can view CTA
- User can view footer

## Tech Stack

- React
- Tailwind CSS
- DaisyUI
- React Toastify
